# DFRobotLCDShield
Implementation of driver for analogic keyboard and LCD 16x2 of DFRobot LCD Shield

* Target board: TM4C1294-EK

* Compiler: Keil MDK

| TM4C1294-EK | DFRobotLCDShield | Function    	        |
|-------------|------------------|----------------------|
| PE0 	      | D4    	         | DB4		              |
| PE1 	      | D5    	       	 | DB5		              |
| PE2 	      | D6    	         | DB6		              |
| PE3 	      | D7    	         | DB7		              |
| PE4 	      | D8    	         | RS		                |
| PE5 	      | D9    	         | EN		                |
| PC4 	      | D10   	         | Back Light	          |
| PD7	        | AD0   	         | Teclado Analogico    |
